==================
Installation guide
==================

Installing habanero
===================

Stable from pypi

.. code-block:: console

    pip install habanero


If you need to fix bibtex format citations using content negotiation use

.. code-block:: console

  pip install habanero[bibtex]


Development version

.. code-block:: console

    pip install git+https://github.com/sckott/habanero.git#egg=habanero
